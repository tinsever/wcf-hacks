# WCF Hacks
## Account Switcher

Für diesen [Account Switcher](https://www.woltlab.com/pluginstore/file/1638-account-switcher-benutzer-verlinken/) gibt es Mobil kein schönes UI, mit dem JS wird das eingebaut. Einfach in den Zeilen 30, 37 & 62 www.your-forum-url.test der URL eures WoltLab Suite Core Pakets ersetzen. Dann den Code als <script> Tag in eurem Seitenheader einbauen.

Falls Hilfe benötigt wird, gerne irgendwo kontaktieren. Ich freue mich natürlich, wenn ihr diese Repository irgendwo erwähnt, ist ja auch MIT lizensiert!
