# WCF Hacks
## Account Switcher

Für diesen [Account Switcher](https://www.woltlab.com/pluginstore/file/1638-account-switcher-benutzer-verlinken/) gibt es Mobil kein schönes UI, mit dem JS wird das eingebaut. Einfach in den Zeilen 45, 53 & 99 www.mein-forum.test/index.php der URL eures WoltLab Suite Core Pakets ersetzen.
Die Suchfunktion lässt sich oben unter Config ausschalten.

Dann den Code als <script> Tag in eurem Seitenheader einbauen.

Falls Hilfe benötigt wird, gerne irgendwo kontaktieren. Ich freue mich natürlich, wenn ihr diese Repository irgendwo erwähnt, ist ja auch MIT lizensiert!
